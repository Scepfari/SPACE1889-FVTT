Suggested modules for SPACE 1889

 

Modul Zweck Foundry URL Project URL

Dice So Nice

 adds the ability to show a 3D dice simulation when a roll is made https://foundryvtt.com/packages/dice-so-nice https://gitlab.com/riccisi/foundryvtt-dice-so-nice

More Coins for DSN

Space 1889 Coins expansion from Dice So Nice

   https://github.com/Scepfari/more-coins-for-dsn

Simple Calendar

a timekeeping module
If active, the time/date is taken over for the time stamp of injuries.

 https://foundryvtt.com/packages/foundryvtt-simple-calendar https://github.com/vigoren/foundryvtt-simple-calendar

SmallTime

 A small module for displaying and controlling the current time of day.  (works together with Simple Calendar) https://foundryvtt.com/packages/smalltime https://github.com/unsoluble/smalltime

Token Action HUD Classic

 This module populates a floating HUD, showing common actions for a controllable token. https://foundryvtt.com/packages/token-action-hud https://github.com/Drental/fvtt-tokenactionhud

Simply Portraits

 add portraits to chat messages https://foundryvtt.com/packages/simply-portraits https://github.com/nickelpro/simply-portraits

Quick Insert - Search Widget

Quickly find characters, monsters, items, spells etc. and insert them wherever you need them right now - the map, character sheets, roll tables, journals, you name it.

 https://foundryvtt.com/packages/quick-insert https://gitlab.com/fvtt-modules-lab/quick-insert

PopOut!

 allows you to open a sheet/application/document into its own window, for easier viewing or for use with multiple monitors https://foundryvtt.com/packages/popout https://github.com/League-of-Foundry-Developers/fvtt-module-popout

Zoom/Pan Options

 Change zooming and panning through the mouse or a touchpad. When zooming in and out, the camera will stay focused on the cursor. https://foundryvtt.com/packages/zoom-pan-options https://github.com/shemetz/ZoomPanOptions

Mass Edit

 Module for searching and editing multiple placeables at the same time. https://foundryvtt.com/packages/multi-token-edit https://github.com/Aedif/multi-token-edit

Advanced Drawing Tools

 Various additional line, fill, and text style options. Multiline text label support. https://foundryvtt.com/packages/advanced-drawing-tools https://github.com/dev7355608/advanced-drawing-tools

Sidebar and Windows Resizer

 * Resize the sidebar horizontally
* Resize the chat text area vertically
* Resize any sidebar floating windows (Combat Tracker, Playlist, etc.)		 https://foundryvtt.com/packages/sidebar-resizer https://github.com/saif-ellafi/foundryvtt-sidebar-resizer

Image Hover

 allows users on the token layer to hover over actor tokens and see the character art https://foundryvtt.com/packages/image-hover https://github.com/Eriku33/Foundry-VTT-Image-Hover

Canvas Fade In and Out

 This module allows for easy hiding of the canvas for players. It has options for setting the background and fade duration to customize the experience. https://foundryvtt.com/packages/canvas-fade-in-out https://github.com/xaukael/canvas-fade-in-out
