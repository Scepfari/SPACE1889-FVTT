Version 2.2.3 -
Decline of achievements

 

v2.2.3

  • System support for Extended Actions => details in help
  • Distance measurement to determine the distance for long-range attacks now take into account the difference in height of the tokens involved
  • localized standard names for items created via the character sheet. For example with german active, a newly created item is given the name "Gegenstand (42)" instead of "new item"
  • Bugfix Sweeping Blow: Internal error if the character size was > 0
  • Bugfix: Specialization "Bombs" in the compendium now has the correct base skill entered
  • Further Foundry V12 preparatory work

 

v2.2.2

  • Space menu revision

    • Clicking on the button no longer switches to a special foundry level, but leaves the current foundry level untouched and opens a floating window containing the familiar menu sub-functions.
    • This means that with foundry V11 it is now once again possible to select tokens before calling one of the functions
    • The position of the Space menu window is memorised by the system so that it appears at the previous position after closing and reopening

    • New "Gravity" entry in the Space menu: This allows you to set the current gravity, which affects the load level of the characters

    • New entry "Disguise token names" in the space menu, which gives selected or all tokens in the scene a generic name if the players have no rights to the tokens. This means that players will not see in the chat or in the encounter list that they are dealing with the right hand of the main villain, for example, but only with NPC 1.

  • Further work on the tooltips

    • Both the tooltips and the chat info creation of items are now based on the same code and differ only in the display which is regulated by css
    • Tooltips of the list items (such as skills, talents, items, weapons, ammunition, money, etc.) are now no longer displayed below the field, but to the left of it.
    • Tooltips for primary and secondary attributes
    • Tooltips also in the NPC and creature sheets
    • Damage tooltip in the same design as the rest
  • Support for fire lances, i.e. the "Fire Lance" weapon, the "FL Energy Storage Crystal" ammunition and the "Fire Lance" rifle specialisation can now be found in the compendiums
  • Foreign language limit now increases further when Linguistics becomes greater than 10, by 4 per level
  • Automatic healing time updates in the character sheet. However, this only works if the players have also been given the right to adjust the time in the Simple Calendar module, otherwise the Simple Calendar will not fire a DateTimeChange event.
  • New archetype "Artefact" added
  • New species "Construct" added
  • In the help document, accessible via the i in the Space menu, information on talent creation has been added, in particular the listing and meaning of the key IDs. Own talents
  • Bugfix: Overload symbol is now also displayed in the NPC sheet
  • Bugfix: The thumbnails of items now retain their original aspect ratio.
  • Bugfix: Distance calculation for ranged attacks, now returns 1.5m instead of 2.12m if the two tokens are diagonally next to each other and the grid size is less than or equal to 1.5m.
  • First internal conversions to become Foundry V12 compatible

  • New buttons in the game settings -> documentation to call up the operating help with a click, as well as to report an error
