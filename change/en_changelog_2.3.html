Version 2.3.5 -
Moon kingdom

2.3.5

  • The token or the character image of a token can now be displayed in large size by pressing a key, the default is [I] or [SHIFT][I]. To do this, simply place the mouse pointer over the desired token beforehand.
  • Bugfixes:
    • When loading and unloading weapons, the type of ammunition is now also shown in the chat
    • The calculation of the gravity penalty in the character sheet has been corrected, whereby the talent Gravity Adaptation is correctly taken into account, i.e. of all home gravity zones, the one closest to the current zone is used for the penalty calculation.
    • Removal of Foundry V12 warnings

 

2.3.4

  • Gravity

    • The home gravity can now be entered for PCs and NPCs
    • A warning appears in the character sheet if the character operates in an unfamiliar gravity field
    • The tooltip provides information about the malus, taking into account the acclimatization time and the talents Gravity Experience and Gravity Adaptation
    • Adaptation of the character in the compendia
  • Revision of the edit / delete icons in the character sheet (style and tooltip)
  • Basic resources can now be edited in the character sheet
  • Revision of the talent sheet: Instead of entering the keys listed in the help, all values can now be easily set via combo boxes
  • New ammunition type sunbeams introduced for weapons that do not require an ammunition item, e.g. like the solar rifle
  • Bugfix: fix Add resource button in NPC sheet
  • Bugfix: talent Directions Sense in the compendium corrected so that the bonus on Orientation is now included

2.3.3

  • Light sources and vision: The system has two new item types (light source, vision improvement) that can be used in the inventory of characters to quickly influence the light and vision settings of tokens. If the Simple Calendar module is active, the settings are automatically reset after the duration of the effect has expired. Details can be found in the help section. Light Sources and Vision
  • Held light sources can be dropped, if they survive the fall undamaged, a light source is placed on the lighting level in the place of the token, which automatically goes out after the remaining illumination time (requires Simple Calendar).
  • Automatic character selection: In encounters, the (NPC) character whose turn it is is automatically selected for the GM. This does not apply to characters that belong to a player. This function can be switched off in the SPACE 1889 settings options.
  • Extended list for the SL in the gravity setting, i.e. it can now be selected in 0.1g steps. In addition, the moons Io, Ganymede and Titan have been added.
  • More information on multiple defences in the chat, i.e. how many defence rolls were made in the round and what penalty it causes.
  • The Add Item selection list is now sorted alphabetically for German. Since the display order is defined via the order in the template.json file, I do not know of a multilingual solution. Sorry, English speakers.

 

2.3.2

  • Bugfix: Missing icons for the music control buttons (#21)
  • Bugfix: Incorrect skill value calculation (#22)
  • Bugfix: No self-defined skills can be selected as pilot skills in the vehicle sheet (#23)
  • Bugfix: The time is not updated when changing combat rounds (#24)
  • Bugfix: Automatic defense: Button missing or not working (#25)
  • Bugfix: Applying first aid does not always work (#26)

 

2.3.1

  • Added new item types for archetype, motivation and species. Items created with these types extend the corresponding selection lists in the character and NPC sheets.
    See: Own archetypes, species and motivations
  • Bugfix: Missing english translation of the weakness flaws

 

2.3.0

  • Support for Foundry VTT V12. The older versions V10 and V11 are still supported.
  • System support for the combat maneuvers Disarm, Grapple and Trip
  • Token HUD extension: When right-clicking on a token, additional buttons are now displayed below the figure. Depending on whether the action is currently possible, these are:
    • Attack with the primary hand weapon
    • Attack with the offhand weapon
    • Grapple
    • Trip
    • Disarm
    • Attack talent, such as Paralyzing Blow or Assassin
    • Any skill roll
  • [Shift] or [CTRL] + left-click on one of these buttons to open the corresponding dialog for specific options and modifiers
  • Adjustment of the tooltip position in the character sheets so that the tooltip is less above the buttons for triggering rolls, especially when the character sheet is near the left edge of the screen.
  • Improved armor tooltips
  • small bugfixes

Note:
Most tests are still on V11, so the test coverage for V12 is in need of improvement, which will not change in the near future as my campaigns are still running on V11. If you find problems with V12, please report them directly on GitHub (a corresponding button can also be found under Game Settings => Report a SPACE 1889 bug) or in Discord on Foundry VTT as a post under GAME SYSTEMS => other-game-systems or directly as a PM to me (Scepfari#1283)
