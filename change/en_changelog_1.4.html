v1.4.0

  • GM screen (new journal compendium SPACE1889 GM info).
  • New Space menu in the left toolbar with three options:
    • Show image in chat (click opens image browser directly [GM only])
    • Show SL screen (click opens the compendium note [GM only])
    • Show help (click opens the help note from the compendium, identical to the chat command: "/help")
  • Foundry Anvil logo extended by Space 1889 in the upper left corner
  • The logo is clickable and opens a window with useful external links, like the GitHub page for submitting bug reports or feature requests, or the link to the Foundry Discord server.
