Version 1.4.3 -
Age of the 1,000 World Wars

 

v1.4.3

  • Display a confirmation dialog when attacking if combat support is active and no target or a dead target is selected.
  • Attack via the Assassin talent is now correctly evaluated as a weapon attack, i.e. the system checks if a melee weapon is held in the hand and uses the one with the most damage considering the Dual Wield talent.
  • Bugfix: SPACE 1889 Wurf macro does not work (#18)

 

v1.4.2

  • Custom SPACE 1889 style for more immersion.
    Can be turned off and on in the game settings
  • Tokens are now subject to movement restrictions if they have at least one of the states Prone, Stunned, Paralyzed, or Unconscious. In this case, the player can no longer move the token until all these states have been removed.
  • The game master can also activate this movement restriction for himself, either in the game settings or faster via the configurable key combination [Ctrl + B]. Regardless of this setting, he can always move figures if they also have the Dead state.
  • Special list type, in which the dots are replaced by Mars spheres, as in this listing. To use it you have to switch to the source code in the editor, find the right place of the enumeration and replace the list tag
    <ul> with
    <ul class="space1889list">

 

v1.4.1

  • Space menu extended by two options:
    • Ready the weapons for all NPCs in a scene, asking whether melee or ranged weapons should be preferred. The weapon with the highest damage is chosen. The corresponding messages in the chat are whispered to the SL. 
    • The display of the life bars and the token names can now be set for all tokens of a scene.
  • When placing tokens on the scene, the system checks if the actor behind the token is already on the scene and if the token is linked to the actor. If this is the case and the token is under the exclusive control of the SL, a dialog appears where the linking can be removed. This also changes the prototype token of the actor.
  • The Foundry pause image has been replaced with an animated one. 

 

v1.4.0

  • GM screen (new journal compendium SPACE1889 GM info).
  • New Space menu in the left toolbar with three options:
    • Show image in chat (click opens image browser directly [GM only])
    • Show SL screen (click opens the compendium note [GM only])
    • Show help (click opens the help note from the compendium, identical to the chat command: "/help")
  • Foundry Anvil logo extended by Space 1889 in the upper left corner
  • The logo is clickable and opens a window with useful external links, like the GitHub page for submitting bug reports or feature requests, or the link to the Foundry Discord server.
