Version 2.1.0 -
Dynasty of the Starborn

v2.1.0

  • Added effect support for attributes. It is now possible to add effects that change the primary and secondary attributes. The necessary attribute keys are listed in the help under Effects.
  • The temporary talent increase per style point can now be triggered directly with a mouse click in the character sheet. The talent is provided with a corresponding effect for this purpose. This function is only available from Foundry VTT V11, V10 does not support the editing of embedded documents in embedded documents. Details here.

  • New attack dialog for modified attacks (requires activated combat support and can be accessed via Shift + mouse click):

    • the attack dialog now contains all options of the total attack as well as the options for auto fire, if supported by the weapon used
    • the respective ammunition consumption is automatically deducted, even for attacks that require more than one bullet
    • the Total Attack as well as the Auto Fire maneuvers now also cause the Total Attack state in which the Active Defense drops to 0. This state disappears automatically when transitioning to the next combat round or when ending the encounter
  • The measuring tool in the UI is now Euclidean, just like the internal distance calculation for firearm attacks
  • Fix: Attacks are only possible if the attacker is a participant in an encounter or if no encounter is active
  • Bugfix: When calculating the loadout, the weapons that are operated via the gunnery skill are no longer included.
  • Bugfix: The worn armor is no longer included twice in the calculation of block, parry and dodge.
  • Bugfix loading encounter values: The number of defenses already performed in the combat round is now retained even after a reload such as a browser F5 refresh.
